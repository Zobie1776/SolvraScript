# cd

## Synopsis
`cd [DIRECTORY]`

## Description
Change the current working directory of the SolvraCLI session. When no directory is specified, `cd` moves to the user's home directory as resolved from the platform.

## Examples
- `cd projects/solvra` — switch into the `projects/solvra` directory relative to the current location.
- `cd` — jump back to the user's home directory.
