# solvra

## Synopsis
`solvra <subcommand>`

## Description
Access Solvra project tools that ship with SolvraCLI. The initial release supports:

- `solvra version` — print the CLI version string.
- `solvra help` — show help for the `solvra` namespace.
- `solvra about` — describe the SolvraCLI goal.

Future releases can extend this namespace without altering the base shell.
