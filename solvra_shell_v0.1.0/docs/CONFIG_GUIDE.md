# Configuration Guide

- Primary config file: `~/.config/solvra-shell/config.toml`
- Themes live under `~/.config/solvra-shell/themes/*.toml`
- Runtime profile overrides can be toggled via `solvractl theme <name>` or `solvractl layout <strategy>`.
- The compositor exposes `config_reload` over JSON-RPC to watch for hot reloads.
